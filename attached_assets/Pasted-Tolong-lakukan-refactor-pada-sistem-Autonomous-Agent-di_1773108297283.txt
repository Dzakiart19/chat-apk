Tolong lakukan refactor pada sistem Autonomous Agent di project ini.

Masalah saat ini:
Agent menampilkan terlalu banyak informasi internal ke UI chat, seperti:

- "Analyzing request..."
- "Searching web..."
- "Extract top 5..."
- "Sending notification..."
- "Wait for user response"
- Catatan sistem seperti "(Note: Since this is a blocking call...)"

Semua hal tersebut adalah proses internal agent dan seharusnya tidak ditampilkan ke user. Saat ini tampilan menjadi berantakan dan terlihat seperti log debugging.

Target yang saya inginkan adalah UI agent yang bersih seperti Manus-style autonomous agent, dimana yang terlihat oleh user hanya:

1. Goal / Tujuan task
2. Progress dalam bentuk card tools
3. Hasil akhir

Instruksi perubahan:

1. Pisahkan tiga layer output agent:
   
   - Internal reasoning (hidden)
   - Tool execution (progress card)
   - Final response (chat message)

2. Semua reasoning agent harus disembunyikan dari UI, termasuk:
   
   - chain-of-thought
   - internal planning
   - debug log
   - system note
   - tool execution raw message

3. Hanya tampilkan progress yang sudah diringkas dalam bentuk tool card, contoh:
   
   🔎 Mencari data framework JavaScript terbaru
   📊 Mengekstrak daftar framework dari hasil pencarian

4. Jangan tampilkan step teknis seperti:
   
   - "Searching web..."
   - "Extracting..."
   - "Analyzing response..."
   
   Semua harus diubah menjadi deskripsi singkat yang ramah pengguna.

5. Format tampilan agent harus menjadi seperti ini:
   
   GOAL
   Menemukan 5 framework JavaScript terbaik tahun 2025.
   
   PROGRESS
   🔎 Mencari ranking framework terbaru
   📊 Mengekstrak framework populer dari hasil pencarian
   
   RESULT
   Berikut 5 framework JavaScript populer di 2025:
   
   1. React
   2. Vue
   3. Angular
   4. Svelte
   5. Next.js

6. Pastikan sistem tidak lagi menampilkan:
   
   - "Wait for user response"
   - "Sending notification"
   - "Analyzing the user's response"
   - atau log agent lainnya.

7. Jika agent menjalankan tools (web search, extraction, API call), tampilkan hanya ringkasan aktivitasnya sebagai progress card, bukan log mentahnya.

8. Pastikan chat UI hanya menerima output yang sudah diformat untuk user.

Goal akhir refactor:
Membuat Autonomous Agent UI terlihat seperti AI assistant profesional (seperti Manus / Devin style agent), bersih, minimal, dan hanya menampilkan informasi yang relevan bagi pengguna.
